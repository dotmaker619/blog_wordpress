== Description ==
Great looking, modern, responsive, Medium styles theme,  Mundana is the perfect choice for your blogging purposes! 
For more information about Mundana please visit https://www.wowthemes.net/themes/mundana-wordpress/.

== Installation ==
1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Upload mundana.zip file.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize > Mundana Theme in your admin panel.
